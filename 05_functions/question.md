# Python Functions: 10 Problems with Solutions 🚀

Here are 10 practice problems to master functions in Python. 

---

<details>
<summary><b>1. Basic Function Syntax</b></summary>
<br>

**Problem:** Write a function to calculate and return the square of a number.

</details>

<details>
<summary><b>2. Function with Multiple Parameters</b></summary>
<br>

**Problem:** Write a function that takes two numbers as parameters and returns their sum.

</details>

<details>
<summary><b>3. Polymorphism in Functions</b></summary>
<br>

**Problem:** Write a function `multiply` that multiplies two numbers, but can also accept and multiply a string and a number (e.g., `multiply('a', 5)` should return `'aaaaa'`).

</details>

<details>
<summary><b>4. Function Returning Multiple Values</b></summary>
<br>

**Problem:** Write a function that returns both the area and circumference of a circle given its radius.

</details>

<details>
<summary><b>5. Default Parameter Value</b></summary>
<br>

**Problem:** Write a function that greets a user. If no name is provided, it should greet with a default name (e.g., "Guest").

</details>

<details>
<summary><b>6. Lambda Function</b></summary>
<br>

**Problem:** Create a lambda function to calculate the cube of a number.

</details>

<details>
<summary><b>7. Function with *args</b></summary>
<br>

**Problem:** Write a function that takes variable number of arguments and returns their sum.

</details>

<details>
<summary><b>8. Function with **kwargs</b></summary>
<br>

**Problem:** Write a function that accepts key-value pairs (keyword arguments) and prints them in the format `key: value`.

</details>

<details>
<summary><b>9. Generator Function with yield</b></summary>
<br>

**Problem:** Write a generator function that yields even numbers up to a specified limit.

</details>

<details>
<summary><b>10. Recursive Function</b></summary>
<br>

**Problem:** Write a recursive function to calculate the factorial of a number.

</details>
