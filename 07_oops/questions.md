# Python Object-Oriented Programming (OOP) Questions 🚗

Here are 10 practice problems to master Object-Oriented Programming in Python.

---

<details>
<summary><b>1. Basic Class and Object</b></summary>
<br>

**Problem:** Create a `Car` class with attributes like `brand` and `model`. Then create an instance of this class.

</details>

<details>
<summary><b>2. Class Method and Self</b></summary>
<br>

**Problem:** Add a method to the `Car` class that displays the full name of the car (brand and model).

</details>

<details>
<summary><b>3. Inheritance</b></summary>
<br>

**Problem:** Create an `ElectricCar` class that inherits from the `Car` class and has an additional attribute `battery_size`.

</details>

<details>
<summary><b>4. Encapsulation</b></summary>
<br>

**Problem:** Modify the `Car` class to encapsulate the `brand` attribute, making it private (`__brand`), and provide a getter method for it.

</details>

<details>
<summary><b>5. Polymorphism</b></summary>
<br>

**Problem:** Demonstrate polymorphism by defining a method `fuel_type` in `Car` and `ElectricCar` that returns different fuel types.

</details>

<details>
<summary><b>6. Class Variables</b></summary>
<br>

**Problem:** Add a class variable to `Car` that keeps track of the number of cars created.

</details>

<details>
<summary><b>7. Static Method</b></summary>
<br>

**Problem:** Add a static method to the `Car` class that returns a general description of what cars are.

</details>

<details>
<summary><b>8. Property Decorators</b></summary>
<br>

**Problem:** Use a property decorator in the `Car` class to make the `model` attribute read-only.

</details>

<details>
<summary><b>9. Class Inheritance and isinstance() Function</b></summary>
<br>

**Problem:** Demonstrate the use of `isinstance()` to check if an instance of `ElectricCar` is also an instance of `Car`.

</details>

<details>
<summary><b>10. Multiple Inheritance</b></summary>
<br>

**Problem:** Create two classes `Battery` and `Engine`, and create an `ElectricCarTwo` class that inherits from both to demonstrate multiple inheritance.

</details>
