# Python Decorator Practice Problems 🎨

Learn about Python Decorators by solving these practice problems.

---

<details>
<summary><b>Problem 1: Timing Function Execution</b></summary>
<br>

**Problem:** Write a decorator that measures the time a function takes to execute.

</details>

<details>
<summary><b>Problem 2: Debugging Function Calls</b></summary>
<br>

**Problem:** Create a decorator to print the function name and the values of its arguments every time the function is called.

</details>

<details>
<summary><b>Problem 3: Cache Return Values</b></summary>
<br>

**Problem:** Implement a decorator that caches the return values of a function, so that when it is called with the same arguments, the cached result is returned instead of re-executing the function.

</details>
